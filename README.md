# MotorPH Payroll System (Object Oriented Programming)

## Overview
This project is a Java-based payroll system developed for MotorPH as part of the course Object Oriented Programming (OOP). The system demonstrates core OOP principles such as encapsulation, modular design, and separation of concerns. It allows users to manage employee data and compute salaries based on attendance records stored in CSV files.

## Features
- View employee records
- Compute employee salary based on selected month
- Read and process attendance data from CSV files
- Modular class structure for payroll processing
- Console-based interface (or GUI if applicable)

## How to Run the Project
### Using NetBeans
1. Open NetBeans
2. Click **Open Project**
3. Select the project folder
4. Run the main class: `OOP_Nodalo.java`

## OOP Concepts Applied
- **Encapsulation** – Data and methods are grouped within classes
- **Modularity** – Separation of payroll logic and data handling
- **Abstraction** – Simplified interface for payroll computation
